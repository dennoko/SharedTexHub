# SharedTexHub

プロジェクト内の共有テクスチャ（MatCap、タイリングテクスチャ、ノーマルマップなど）を一元管理し、簡単にアクセス・再利用できるようにするUnityエディタ拡張です。

## 機能
- **自動検出**: マテリアル（lilToon）をスキャンして、使用されているMatCapやタイリングテクスチャを自動的に検出します。
- **手動登録**: 右クリックメニューや専用フォルダへのドラッグ＆ドロップで、任意のテクスチャをライブラリに追加できます。
- **スマートな整理**: MatCap、Tiling、Normal、Mask、Decalのタブに自動分類します。
- **カラーソート**: 色相や色のばらつきでソートし、視覚的に探しやすくします。
- **ドラッグ＆ドロップ**: Hubからマテリアルスロットやシーンオブジェクトへ直接テクスチャをドラッグ＆ドロップできます。

## 動作要件
- Unity 2019.4 以降
- lilToon シェーダー（自動検出機能に推奨）

## インストール方法
1. Unityプロジェクトの `Assets` フォルダにパッケージをインポートしてください。
2. メニューバーの `Tools > SharedTexHub` からツールを使用できます。
